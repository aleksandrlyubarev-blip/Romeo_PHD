# RoboQC

**Контроль качества от NPI до масштаба.**

RoboQC — направление **NVLabs**, развивается как часть **RomeoFlexVision**.

Это концепт-пакет продукта: материалы для питча инвесторам и ранних продаж клиентам.
Папка самодостаточна — её можно вынести в отдельный репозиторий, когда появится
постоянное имя/домен (`git subtree split --prefix=roboqc` или простое копирование).

## Структура

```
roboqc/
├── README.md              # этот файл
├── docs/
│   ├── concept.md         # концепция: проблема, решение, клиенты, дорожная карта
│   ├── architecture.md    # путь NPI → масштаб и стек NVIDIA (Mermaid-диаграммы)
│   └── ad-script.md       # сценарий рекламного ролика (~80 сек)
├── landing/
│   └── index.html         # лендинг-раздел в стилистике RomeoFlexVision / NVLabs
└── visuals/
    ├── roboqc-dashboard.png   # мокап интерфейса
    └── roboqc-scanner.png     # ручной сканер ошибок в работе
```

## Статус

Временное имя проекта — **NVLabs**. Постоянный бренд/домен определяется позже
(ориентир — сентябрь, либо при появлении инвестора). До этого веб-присутствие
RoboQC живёт как раздел сайта RomeoFlexVision.

> Замечание по бренду: имя «NVLabs» близко к исследовательскому подразделению
> NVIDIA (`NVlabs`). Для внутреннего/питч-использования допустимо, но перед
> публичным запуском имя стоит проверить на товарный знак.
